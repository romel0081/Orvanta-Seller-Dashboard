Connect to Supabase

Your data will be stored securely in your own Supabase database. This only takes 2 minutes to set up.

1. Go to supabase.com → New project
2. Settings → API → copy Project URL and anon key
3. Go to SQL Editor → run the setup SQL (see SETUP.md)
4. Paste your credentials below

Credentials are stored locally in your browser only.
